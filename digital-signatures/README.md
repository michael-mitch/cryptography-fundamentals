# Digital Signatures

This section demonstrates authentication and integrity verification concepts using digital signatures and elliptic curve cryptography.

## Concepts Demonstrated
- Digital signatures
- ECDSA concepts
- Elliptic Curve Cryptography (ECC)
- Authentication
- Integrity verification

## Skills Demonstrated
- Signature verification concepts
- Understanding authenticity and integrity
- Public-key validation methods
- ECC-based cryptographic reasoning

## Overview
These exercises demonstrate how digital signatures verify that a message was created by a trusted sender and has not been modified during transmission.
