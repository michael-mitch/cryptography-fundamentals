# Exams and Practical Cryptography Analysis

This section contains practical cryptography exercises and applied assessment work focused on encryption, decryption, authentication, secure key exchange, and digital signature concepts.

## Topics Covered
- Frequency analysis
- Caesar and affine ciphers
- XOR operations
- Stream ciphers
- AES concepts
- Diffie-Hellman key exchange
- ElGamal encryption
- Elliptic Curve Cryptography (ECC)
- ECDSA digital signatures
- Authentication and integrity verification
- Modular arithmetic
- Confusion and diffusion
- DES limitations

## Skills Demonstrated
- Manual cryptographic analysis
- Encryption and decryption workflows
- Public-key cryptography concepts
- Digital signature verification
- Secure key exchange reasoning
- Cipher breaking techniques
- Applied cryptography problem solving

## Overview
These exercises demonstrate hands-on application of classical and modern cryptographic concepts through practical analysis, mathematical reasoning, and secure communication scenarios.
