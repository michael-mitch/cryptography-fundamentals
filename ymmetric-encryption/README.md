# Symmetric Encryption

This section explores shared-key encryption concepts and demonstrates how encryption and decryption are performed using the same secret key.

## Concepts Demonstrated
- Shared-key cryptography
- Encryption and decryption workflows
- Modular arithmetic
- Key-based confidentiality

## Skills Demonstrated
- Understanding symmetric encryption principles
- Manual encryption/decryption techniques
- Secure communication concepts

## Overview
These exercises focus on how symmetric encryption protects information by requiring both parties to use the same secret key during communication.
