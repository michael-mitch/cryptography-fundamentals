# Cryptography Fundamentals

This repository contains hands-on cryptography exercises and security analysis projects covering encryption, decryption, hashing, digital signatures, and secure communication concepts.

## Skills Demonstrated
- Manual encryption and decryption
- Symmetric encryption concepts
- Asymmetric encryption concepts
- Diffie-Hellman key exchange
- ElGamal encryption
- Elliptic Curve Cryptography (ECC)
- Digital signature verification
- Hashing and integrity concepts
- Secure communication analysis

## Topics Covered

### Classical Ciphers
Explored foundational encryption techniques including rotation and substitution-based ciphers to understand core cryptographic principles.

### Symmetric Encryption
Worked with shared-key encryption concepts and the process of encrypting and decrypting data using common cryptographic methods.

### Asymmetric Encryption
Applied public-key cryptography concepts including Diffie-Hellman key exchange and ElGamal encryption.

### Digital Signatures
Demonstrated authentication and integrity verification concepts using ECDSA and elliptic curve cryptography techniques.

### HTTPS/TLS Security Analysis
Analyzed how cryptographic protocols are used within HTTPS/TLS to provide confidentiality, integrity, authentication, and secure communication across networks.

## Tools & Concepts Used
- Diffie-Hellman Key Exchange
- ElGamal Encryption
- Elliptic Curve Cryptography (ECC)
- ECDSA
- Hashing Concepts
- HTTPS/TLS

## Notes
These exercises and projects were completed as part of hands-on cryptography and cybersecurity training to build a foundational understanding of modern encryption and authentication methods.
