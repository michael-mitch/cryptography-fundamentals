# Asymmetric Encryption

This section focuses on public-key cryptography concepts including Diffie-Hellman key exchange and ElGamal encryption.

## Concepts Demonstrated
- Public and private keys
- Diffie-Hellman key exchange
- ElGamal encryption
- Shared secret generation
- Modular exponentiation

## Skills Demonstrated
- Understanding asymmetric cryptography
- Secure key exchange concepts
- Public-key encryption workflows
- Encryption and decryption analysis

## Overview
These exercises demonstrate how secure communication can occur without directly transmitting secret keys and how public-key systems are used to protect data.
