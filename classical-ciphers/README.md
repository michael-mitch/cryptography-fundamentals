# Classical Ciphers

This section focuses on foundational cryptographic techniques used to demonstrate the basic concepts of encryption and decryption.

## Concepts Demonstrated
- Caesar/rotation ciphers
- Substitution-based encryption
- Manual encryption and decryption
- Character shifting and modular arithmetic

## Skills Demonstrated
- Understanding of early cryptographic methods
- Encryption and decryption logic
- Step-by-step cryptographic reasoning

## Overview
These exercises demonstrate how plaintext can be transformed into ciphertext using classical encryption methods and how the original message can be recovered using the correct key.
